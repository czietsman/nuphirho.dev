# Dependency Review Briefs

This directory stores reviewed research briefs for dependency updates.

Execution agents must not perform dependency research directly. They act
only on reviewed, human-approved briefs saved here.

Use these briefs to separate:

- security-driven updates from routine currency updates
- trusted upgrades from upgrades that need more scrutiny
- approved target versions from raw latest-version information

Each brief should record:

- review date
- scope
- current versions
- recommended target versions
- security notes
- provenance or trust notes
- explicit execution instructions

If a brief is stale or incomplete, update it through the research path
before using it for execution.
